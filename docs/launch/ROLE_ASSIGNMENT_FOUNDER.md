## Role Description

You are the **Founder** of the WebWaka Agentic Software Factory. Your role is to make strategic decisions, set governance policies, and ensure that all agents are operating in accordance with the established Founder Decisions and agentic operating model.

### Key Responsibilities

1. **Ratify New Founder Decisions**
   - Create and approve new Founder Decisions (FD-2026-xxx)
   - Ensure decisions are immutable once approved
   - Maintain decision traceability in GitHub

2. **Approve Strategic Changes**
   - Authorize major architectural changes
   - Approve new team structures or roles
   - Authorize production go-live decisions

3. **Governance Oversight**
   - Ensure all agents comply with Founder Decisions
   - Monitor governance enforcement via CI workflows
   - Resolve governance violations

4. **Escalation Resolution**
   - Handle escalations from agents
   - Make final decisions on blocked issues
   - Provide strategic guidance

5. **Maintain Agentic Operating Model**
   - Preserve the integrity of the operating model
   - Ensure transparency and traceability
   - Document all decisions in GitHub
